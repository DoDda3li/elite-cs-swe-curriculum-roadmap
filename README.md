# Elite CS + SWE Curriculum
### منهج علوم الحاسوب وهندسة البرمجيات المتقدم

> A self-directed, bilingual (Arabic + English) Computer Science and Software Engineering curriculum spanning 12 phases — designed to exceed the depth of MIT, Harvard, and Stanford programs.
>
> منهج ذاتي التعلم، ثنائي اللغة (عربي + إنجليزي)، يغطي 12 مرحلة — مصمم ليتجاوز عمق برامج MIT وهارفارد وستانفورد.

## Graduation Requirements | متطلبات التخرج

- [ ] Cumulative GPA ≥ 2.7 across all 12 phases
- [ ] 300+ LeetCode problems solved — all 20 patterns mastered
- [ ] All 12 phases completed with weighted score ≥ 60% each
- [ ] Bilingual notes (Arabic + English) maintained throughout
- [ ] "Why I Started" document written on Day 1 and reviewed each phase
- [ ] 3 major projects peer-reviewed publicly (P5, P7, P11)
- [ ] P11 capstone distributed system deployed to production
- [ ] Technical blog post published bilingually (Arabic + English)
- [ ] Arabic blog maintained throughout the curriculum

## Academic Policies | السياسات الأكاديمية

**Grading weights per phase:**
| Exam Type | Weight | Policy |
|---|---|---|
| Daily | 10% | 5 per study week (weekdays). Missing = 0 |
| Weekly | 15% | 1 per study week. Missing = 0 |
| Midterm | 25% | 1 per 5 study weeks. Missing = must retake before advancing |
| Final | 40% | 1 per phase. Missing = cannot graduate phase |
| Project | 10% | 1 per phase. Missing = cannot graduate phase |

**Grade scale:** S (95+) · A+ (90) · A (85) · A− (80) · B+ (76) · B (72) · B− (68) · C+ (64) · C (60) · D (50) · F (<50)

**Key rules:**
- Late exams: no penalty — take whenever
- Retakes: best score always kept
- Phase score < 60%: retake final before advancing
- GPA < 2.0 in any phase: mandatory phase review
- Anki: daily review required — missing 3+ days triggers retake
- Bilingual: all notes, Anki cards, READMEs in Arabic + English
- Rest weeks: mandatory after each phase — no new material

### P0 — Computing Foundations | أساسيات الحوسبة
**Dates:** May 3 – Jul 4, 2026 · **Weeks:** 6 study + 1 bilingual + 1 rest · **Hours:** ~90

How computers work, internet protocols, Linux terminal, Git, VS Code, algebra and math readiness. The entry point — no prior experience required.

| Week | Topic |
|---|---|
| W1 | How computers work — binary, ASCII, CPU (CS50x Week 0) |
| W2 | The internet — HTTP, DNS, TCP/IP |
| W3 | Linux command line and terminal mastery |
| W4 | Version control — Git and GitHub |
| W5 | Editors, debugging and regex |
| W6 | Math readiness + P0 capstone portfolio |
| B | Bilingual consolidation — rewrite all notes in Arabic |
| R | 🌙 Rest week |

**Resources:**
- [Harvard CS50x](https://cs50.harvard.edu/x/) — Best CS introduction, free
- [MIT Missing Semester](https://missing.csail.mit.edu/) — Terminal, shell, Git, debugging
- [Khan Academy Computing](https://www.khanacademy.org/computing) — How computers and internet work
- [Pro Git Book](https://git-scm.com/book/en/v2) — Definitive Git reference, free

### P1 — Mathematical Foundations | الأسس الرياضية
**Dates:** Jul 5 – Sep 23, 2026 · **Weeks:** 12 study + 2 bilingual + 1 rest · **Hours:** ~180

Proof-writing, combinatorics, number theory, graph theory, linear algebra (eigenvalues, SVD, PCA), probability (Bayes, distributions, CLT).

| Weeks | Module |
|---|---|
| W1–5 | Discrete Mathematics — logic, induction, sets, number theory, combinatorics |
| W6–9 | Linear Algebra — vectors, transformations, eigenvalues, SVD |
| W10–12 | Probability & Statistics — Bayes, distributions, CLT |

**Resources:**

- [Kimberly Brehm — Discrete Math YouTube](https://www.youtube.com/playlist?list=PLl-gb0E4MII28GykmtuBXNUNoej-vY5Rz) — Best visual explanations
- [3Blue1Brown — Essence of Linear Algebra](https://www.youtube.com/playlist?list=PLZHQObOWTQDPD3MizzM2xVFitgF8hE_ab) — Geometric intuition

- [Harvard Stat 110 — Blitzstein](https://www.youtube.com/playlist?list=PL2SOU6wwxB0uwwH80KTQ6ht66KWxbzTIo) — Best probability course
