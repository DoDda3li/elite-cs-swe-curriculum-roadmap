## Academic Policies | السياسات الأكاديمية

Grading weights per phase (total = 100%):

| Exam Type | Weight | Policy |
|---|---|---|
| **Daily** | 10% | 5 per study week (weekdays). Missing = 0 |
| **Weekly** | 15% | 1 per study week. Missing = 0 |
| **Midterm** | 25% | 1 per 5 study weeks. Missing = must retake before advancing |
| **Final** | 40% | 1 per phase. Missing = cannot graduate phase |
| **Project** | 10% | 1 per phase. Missing = cannot graduate phase |

**Grade scale (4.30):** S (95+) · A+ (90) · A (85) · A− (80) · B+ (76) · B (72) · B− (68) · C+ (64) · C (60) · D (50) · F (<50)

**Key rules:**
- Late exams: no penalty — take whenever
- Retakes: best score always kept
- Phase score < 60%: retake final before advancing
- GPA < 2.0 in any phase: mandatory phase review
- Anki: daily review required — missing 3+ days triggers retake
- Bilingual: all notes, Anki cards, READMEs in Arabic + English
- **Rest weeks**: mandatory after each phase — no new material
- **Buffer weeks** (NEW): one week between each pair of phases — absorbs delays, illness, life events without breaking schedule

## The 12 Phases | المراحل الاثنتا عشرة

### P0 — Computing Foundations | أساسيات الحوسبة
**Dates:** May 3 – Jul 4, 2026 · **Weeks:** 6 study + 1 bilingual + 1 rest · **Hours:** ~90 · **Exams:** 39

How computers work, internet protocols, Linux terminal, Git, VS Code, algebra and math readiness. The entry point — no prior experience required.

| Week | Topic |
|---|---|
| W1 | How computers work — binary, ASCII, CPU (CS50x Week 0) |
| W2 | The internet — HTTP, DNS, TCP/IP |
| W3 | Linux command line — **Midterm exam covers W1–W3** |
| W4 | Version control — Git and GitHub |
| W5 | Editors, debugging, regex |
| W6 | Math readiness + P0 capstone — **Final exam covers all 6 weeks** |
| B | Bilingual consolidation |
| R | 🌙 Rest week |

**Resources:** Harvard CS50x · MIT Missing Semester · Khan Academy Computing · Pro Git Book
